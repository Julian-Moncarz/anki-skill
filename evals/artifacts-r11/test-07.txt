

Here's a function that reverses a linked list in Python:

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverse_linked_list(head: ListNode) -> ListNode:
    prev = None
    curr = head
    while curr:
        next_node = curr.next
        curr.next = prev
        prev = curr
        curr = next_node
    return prev
```

The algorithm iterates through the list, reversing each node's `next` pointer to point to the previous node instead of the next one. It uses three pointers (`prev`, `curr`, `next_node`) and runs in O(n) time with O(1) space.
