

```python
def reverse_linked_list(head):
    prev = None
    curr = head
    while curr:
        next_node = curr.next
        curr.next = prev
        prev = curr
        curr = next_node
    return prev
```

This iteratively walks the list, flipping each node's `next` pointer to point backward. Three pointers track the previous, current, and next nodes. Returns `prev` — the new head (old tail).
