

Here's a function that reverses a linked list in Python:

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


def reverse_linked_list(head: ListNode) -> ListNode:
    prev = None
    curr = head
    while curr:
        nxt = curr.next
        curr.next = prev
        prev = curr
        curr = nxt
    return prev
```

**How it works:**

1. Maintain three pointers: `prev`, `curr`, and `nxt`
2. At each step, save the next node, reverse the current node's pointer to point backward, then advance both `prev` and `curr`
3. When `curr` is `None`, `prev` points to the new head

**Time complexity:** O(n)
**Space complexity:** O(1)

**Example usage:**

```python
# Build: 1 -> 2 -> 3 -> None
head = ListNode(1, ListNode(2, ListNode(3)))

reversed_head = reverse_linked_list(head)

# Result: 3 -> 2 -> 1 -> None
node = reversed_head
while node:
    print(node.val, end=" -> ")
    node = node.next
print("None")
```
