**About your actual error: `TypeError: 'NoneType' object is not iterable`**

This error occurs when you try to iterate over (loop through, unpack, etc.) a value that is `None`. Common causes:

1. **Function returning `None` implicitly:**
   ```python
   def get_items():
       items = [1, 2, 3]
       # forgot to return items!
   
   for item in get_items():  # TypeError: NoneType is not iterable
       print(item)
   ```

2. **Unpacking a `None` return:**
   ```python
   a, b = some_function()  # fails if some_function() returns None
   ```

3. **Method that modifies in-place returning `None`:**
   ```python
   items = [3, 1, 2].sort()  # .sort() returns None, sorts in-place
   for item in items:         # TypeError
       ...
   ```
