

Here's a simple linked list reversal in Python:

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def reverse_linked_list(head: ListNode) -> ListNode:
    prev = None
    curr = head
    while curr:
        next_node = curr.next
        curr.next = prev
        prev = curr
        curr = next_node
    return prev
```

The algorithm iterates through the list, reversing each node's `next` pointer to point to the previous node instead. Three pointers track the previous, current, and next nodes. Time complexity is O(n), space is O(1).
